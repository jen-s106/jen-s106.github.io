Fighting Against Deforestation

For my semester-long project, I wanted to expand on a project idea that I had in my Digital Design class. The assignment was to create a prototype for a campaign we want our audience to become aware of, I chose deforestation. I chose to create an app that will engage the user to learn more about the different types of trees that they encounter, this will work when the user takes a picture of a tree. The user will be able to learn about the problems that deforestation causes, and different charities.
